# AGENTS.md — Глобальный регламент разработки

Этот файл определяет архитектуру, бизнес-логику и общие правила проекта. Перед началом работы над конкретным слоем приложения, обязательно изучите соответствующее руководство:

## Все настройки агента только в папке ./.opencode

**Главные правила требующие прочтения данного файа ./.opencode/AGENTS.md**

- Запрос начинается с ключевых слов `Новая задача` - ты обязан полностью прочитать файл и исследовать все ссылки из данного файла, для добавления в контекст нужной информации, полученной по контексту поставленной задачи
- Если в запросе пользователя есть ключевые слова [`documents_hub, contextDocs`] - ты обязан полностью прочитать файл и исследовать все ссылки из данного файла
