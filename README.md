# assets_monitoring

Monitoring service with be, fe, and db

# Мой план на данный проект:

- Разобраться со spec-kit - установить его, понять как им пользоваться, начать с его помощью проектирование приложения
- Приложение будет состоять из сервера, который обязан выполнять все обработки и запросы к api различных бирж, производить вычисления и отдавать данные на клиента
- Клиент - нужен для визуализации и редактирования портфеля, хочется дать возможность добавления вкладок под конкретного пользователя, у вкладок будут свои шаблоны, под фонду, под токены и под nft
- Система авторизации и аутентификации, понять весь этот механизм, как создавать учетки и роли, как хранить, и как сделать данный сервис безопасным, или использовать уже какие-то готовые наработки
- База данных - хранит данные о пользователе и все его вкладки с настройками
