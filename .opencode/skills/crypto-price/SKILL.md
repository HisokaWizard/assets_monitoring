# Crypto Price Tool

Получение текущих цен криптовалют через CoinGecko API.

## Когда использовать

- Нужно узнать текущую цену ATOM, MOVE или других токенов
- Анализ портфеля криптовалют
- Обновление данных об активах

## Подход

1. Проверить кэш (данные кэшируются на 60 секунд)
2. Если нет в кэше — запросить из CoinGecko API
3. Обработать ответ и отформатировать
4. Вернуть результат с метаданными

## Примеры

**Простой запрос:**
```
Получи цену ATOM
→ ATOM: $4.23
```

**Детальная информация:**
```
Получи детальную информацию о MOVE
→ MOVE: $0.85
   Market Cap: $1.24B
   24h Volume: $45.67M
   24h Change: +5.23%
```

## Ограничения

- Поддерживает только токены из списка: ATOM, MOVE, BTC, ETH, SOL, AVAX, DOT, LINK
- Rate limit от CoinGecko API
- Кэш действует 60 секунд
