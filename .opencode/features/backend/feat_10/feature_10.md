# Feature 10: Безопасность JWT — убрать fallback секрета

## Описание
Критическая уязвимость безопасности в `backend/src/auth/auth.module.ts:35`. При отсутствии переменной окружения `JWT_SECRET` используется тривиальный fallback `'secret'`, который злоумышленник может легко угадать для генерации своих JWT токенов с любой ролью.

**Решение:** При отсутствии `JWT_SECRET` приложение должно падать с понятной ошибкой при старте.

Backend
