# Sub-task 2: Создание первой миграции

## Описание
Создать первую миграцию на основе существующих entities.

## Способ решения

### 1. Сгенерировать миграцию
```bash
npm run migration:generate -- src/database/migrations/InitialSchema
```

### 2. Применить миграцию
```bash
npm run migration:run
```

## Критерии приёмки
- [ ] Миграция создана
- [ ] Все таблицы созданы
- [ ] Приложение работает
