# Sub-task 2: Создание package.json

## Задача

Создать package.json с зависимостями и скриптами для проекта.

## Способ решения

Определить production и development зависимости согласно стеку:
- React 18+, React DOM
- TypeScript 5+
- Redux Toolkit, RTK Query
- React Router 6
- MUI 7
- Webpack 5 + loaders

## Ожидаемый результат

Файл package.json с полным набором зависимостей.

## Критерии приёмки

- [ ] package.json содержит все необходимые зависимости
- [ ] Скрипты настроены: dev, build, test
- [ ] Версии библиотек соответствуют требованиям
