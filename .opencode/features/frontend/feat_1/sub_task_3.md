# Sub-task 3: Создание tsconfig.json

## Задача

Настроить TypeScript конфигурацию с strict mode и путями.

## Способ решения

- Включить strict: true
- Настроить baseUrl и paths для алиасов
- Указать целевую версию ES
- Настроить jsx

## Ожидаемый результат

Файл tsconfig.json с корректной конфигурацией.

## Критерии приёмки

- [ ] strict mode включен
- [ ] Настроены paths для FSD слоев
- [ ] jsx режим настроен для React 18+
