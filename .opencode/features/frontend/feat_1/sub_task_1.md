# Sub-task 1: Создание структуры директорий

## Задача

Создать базовую файловую структуру проекта frontend согласно методологии FSD (Feature-Sliced Design).

## Способ решения

Создать следующую иерархию директорий:

```
frontend/
├── src/
│   ├── app/
│   │   ├── providers/
│   │   └── styles/
│   ├── pages/
│   ├── widgets/
│   ├── features/
│   ├── entities/
│   └── shared/
│       ├── api/
│       ├── ui/
│       ├── lib/
│       └── config/
├── public/
└── tests/
    ├── unit/
    └── e2e/
```

## Ожидаемый результат

Все директории созданы и готовы к добавлению файлов.

## Критерии приёмки

- [ ] Существуют все директории согласно структуре
- [ ] В каждой директории создан placeholder файл .gitkeep
