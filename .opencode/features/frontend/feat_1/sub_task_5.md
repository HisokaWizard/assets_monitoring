# Sub-task 5: Создание базовых файлов приложения

## Задача

Создать точку входа, App компонент и базовые провайдеры.

## Способ решения

- Создать index.tsx
- Создать App.tsx с базовой структурой
- Настроить провайдеры (Redux, Router, MUI Theme)

## Ожидаемый результат

Приложение запускается и отображает базовый UI.

## Критерии приёмки

- [ ] index.tsx создает root и рендерит App
- [ ] App.tsx содержит базовую разметку
- [ ] Провайдеры настроены корректно
