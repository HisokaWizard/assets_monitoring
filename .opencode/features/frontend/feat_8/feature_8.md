# Feature 8: NFT Portfolio Page

## Описание

Реализация страницы NFT портфеля с редактируемой таблицей, возможностью добавления новых NFT с автоматическим получением цены, обновлением котировок и генерацией отчетов.

## Контекст

- Страница TokensPage уже реализована и используется как паттерн
- API для активов уже поддерживает NFT (тип 'nft')
- AssetUpdateService содержит логику для получения цен NFT из OpenSea
- Необходимо добавить метод для получения цены NFT при создании

## Декомпозиция

### 1. Backend: Добавить метод получения цены NFT (sub_task_1)
- Добавить метод getNFTPrice в AssetsService
- Модифицировать метод create для вызова getNFTPrice при создании NFT
- Рассчитать все расчетные поля при создании

### 2. Backend: Добавить endpoint для refresh NFT (sub_task_2)
- Добавить новый endpoint POST /assets/refresh-nft
- Реализовать логику обновления только NFT активов

### 3. Frontend: Обновить assetsApi (sub_task_3)
- Добавить метод refreshNFTs для вызова нового endpoint
- Экспортировать типы NFTAsset

### 4. Frontend: Реализовать страницу NFT (sub_task_4)
- Реализовать редактируемую таблицу по аналогии с TokensPage
- Добавить форму добавления/редактирования NFT
- Добавить кнопки: Update Quotes, Generate Report
- Реализовать обновление при загрузке страницы
- Отобразить все поля из NFTAsset DTO

## Зависимости

- Требуется существующий AssetsService и AssetUpdateService
- Требуется страница TokensPage как референс

## Оценка сложности

Medium - работа по готовому паттерну
