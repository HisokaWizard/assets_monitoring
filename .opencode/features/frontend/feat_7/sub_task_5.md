# Sub-task 5: Реализовать TokensPage с редактируемой таблицей

## Описание

Обновить страницу TokensPage (/tokens) с полным функционалом:
- Отображение всех токенов пользователя в таблице
- Редактируемые ячейки (тикер, количество, средняя цена)
- Форма добавления нового токена (модальное окно)
- Кнопка "Обновить котировки"
- Автоматическое обновление при монтировании компонента

## Способ решения

1. Использовать Material UI Table для отображения
2. Использовать RTK Query хуки из sub-task 4
3. Добавить модальное окно для создания токена
4. Добавить useEffect для автоматического обновления

## Подготовка тесткейсов для TDD

1. **Тест: Отображение таблицы токенов**
   - Компонент: TokensPage
   - Ожидание: отображает таблицу с данными

2. **Тест: Добавление нового токена**
   - Пользователь вводит: BTC, 1, 50000
   - Ожидание: токен добавлен, таблица обновлена

3. **Тест: Обновление котировок**
   - Пользователь нажимает "Обновить котировки"
   - Ожидание: данные обновлены, отображается уведомление

4. **Тест: Автоматическое обновление**
   - Пользователь переходит на /tokens
   - Ожидание: данные загружаются автоматически

## Ожидаемый результат

TokensPage содержит:
- Таблицу с колонками: Тикер, Название, Количество, Средняя цена, Текущая цена, Стоимость, P/L, %
- Форму добавления токена
- Кнопку обновления котировок
- Автоматическую загрузку данных
