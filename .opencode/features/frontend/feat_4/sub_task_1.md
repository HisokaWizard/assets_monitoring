# Sub-task 1: UI компонент RegistrationPage + unit тесты

## Описание задачи

Создать UI компонент страницы регистрации с формой и валидацией.

## Способ решения

Использовать документацию:
- REACT.md - функциональные компоненты, hooks
- MUI.md - компоненты форм
- TESTING.md - тестирование компонентов

Компонент должен:
- Использовать те же стили и паттерны, что и LoginPage
- Включать поля: email, password, confirm password, role (select)
- Использовать useState для управления формой
- Иметь валидацию на клиенте

## Подготовка тесткейсов для TDD

### Unit тесты (до реализации - должны упасть)

1. **render_registration_form** - Компонент отображает форму регистрации
   - Ожидание: отображаются поля email, пароль, подтверждение пароля, роль
   - Ожидание: отображается кнопка "Зарегистрироваться"

2. **validation_empty_fields** - Валидация при пустых полях
   - Ожидание: показывается ошибка "Email обязателен"
   - Ожидание: показывается ошибка "Пароль обязателен"
   - Ожидание: показывается ошибка "Подтверждение пароля обязательно"

3. **validation_invalid_email** - Валидация некорректного email
   - Ожидание: показывается ошибка "Некорректный email"

4. **validation_password_mismatch** - Валидация несовпадения паролей
   - Ожидание: показывается ошибка "Пароли не совпадают"

5. **validation_password_length** - Валидация длины пароля
   - Ожидание: показывается ошибка "Пароль должен быть минимум 6 символов"

6. **show_loading_state** - Отображение состояния загрузки
   - Ожидание: кнопка disabled во время загрузки
   - Ожидание: отображается индикатор загрузки

7. **show_error** - Отображение ошибки
   - Ожидание: отображается Alert с текстом ошибки

## Ожидаемый результат

- Файл: frontend/src/pages/register/RegisterPage.tsx
- Файл: frontend/src/pages/register/__tests__/RegisterPage.test.tsx
- Все тесты проходят

## Критерии приёмки

1. Компонент отображает форму с 4 полями
2. Валидация работает для всех полей
3. Состояние загрузки отображается корректно
4. Ошибки отображаются в Alert компоненте
5. 7 unit тестов проходят
