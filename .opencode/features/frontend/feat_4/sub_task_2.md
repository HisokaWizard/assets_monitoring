# Sub-task 2: Роутинг + навигация

## Описание задачи

Добавить маршрут для страницы регистрации и ссылку на неё со страницы логирования.

## Способ решения

Использовать документацию:
- ROUTING.md - React Router 6

Изменения:
1. Добавить маршрут /register в router/index.tsx
2. Добавить ссылку "Зарегистрироваться" в LoginPage
3. Использовать useNavigate для программной навигации

## Подготовка тесткейсов для TDD

### Unit тесты (до реализации - должны упасть)

1. **route_exists** - Маршрут /register существует
   - Ожидание: router содержит маршрут с path='/register'

2. **navigation_to_register** - Навигация на страницу регистрации
   - Ожидание: при клике на ссылку происходит переход на /register

3. **login_page_has_link** - Страница логирования содержит ссылку на регистрацию
   - Ожидание: отображается ссылка "Зарегистрироваться"
