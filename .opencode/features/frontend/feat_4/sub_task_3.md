# Sub-task 3: Интеграция с useRegister + интеграционные тесты

## Описание задачи

Интегрировать RegistrationPage с useRegister hook и добавить интеграционные тесты.

## Способ решения

Использовать:
- useRegister hook (существующий)
- authApi (существующий)
- navigate для редиректа

Логика:
1. При успешной регистрации - редирект на /login
2. При ошибке - показать ошибку в Alert

## Подготовка тесткейсов для TDD

### Интеграционные тесты (до реализации - должны упасть)

1. **successful_registration_redirect** - Успешная регистрация редиректит на /login
   - Действие: заполнить форму валидными данными
   - Действие: отправить форму
   - Ожидание: navigate вызван с '/login'

2. **registration_error** - Ошибка регистрации отображается
   - Действие: отправить форму с данными, вызывающими ошибку
   - Ожидание: отображается error Alert

3. **registration_success_with_user_data** - Успешная регистрация с данными пользователя
   - Действие: заполнить форму с email, password, role
   - Ожидание: useRegister вызывается с правильными данными
