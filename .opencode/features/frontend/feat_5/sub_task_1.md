# Sub-task 1: Создание страниц-заготовок

## Описание

Создать пустые страницы-заготовки для NFT, Tokens и Profile.

## Способ решения

Следовать паттерну существующих страниц (login, register):
- `frontend/src/pages/{name}/{PageName}.tsx` - компонент страницы
- `frontend/src/pages/{name}/index.ts` - экспорт

Страницы должны содержать базовую структуру с MUI Container и Typography.
