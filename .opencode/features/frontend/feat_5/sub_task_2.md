# Sub-task 2: Обновление HomePage с навигационными карточками

## Описание

Добавить на HomePage 4 навигационные карточки (MUI Card) со ссылками:
- NFTs → /nfts
- Tokens → /tokens
- Profile → /profile
- Logout → logout action

## Способ решения

Использовать MUI компоненты:
- `Card`, `CardContent`, `CardActionArea` для карточек
- `Grid` или `Box` для раскладки
- `Typography` для заголовков
- `Link` или `useNavigate` для навигации

Стилизация: использовать MUI sx prop, следовать существующему стилю кода.
