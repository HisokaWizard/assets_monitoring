# Feature 1: Деплой и настройка продакшена на Raspberry Pi 4

## Описание задачи
Настроить production-деплой приложения на Raspberry Pi 4 с бесплатным доменом, включая настройку веб-сервера, SSL, firewall и автоматического деплоя.

## Контекст

**Технологический стек:**
- Frontend: React (webpack)
- Backend: NestJS (Node.js)
- Database: SQLite
- ОС: Raspberry Pi 4 (Raspberry OS / Ubuntu)

**Требования:**
- Продакшен с SSL (Let's Encrypt)
- Firewall (w)
- Автозапуск серufвисов
- Автоматический деплой через CI/CD
- Бесплатный домен (.tk, .ml, .ga, .cf, .gq)

## Декомпозиция

1. **Настройка инфраструктуры** — установка и настройка nginx, Node.js, firewall
2. **Конфигурация Backend** — настройка PM2 для автозапуска, переменные окружения
3. **Конфигурация Frontend** — сборка production билда, конфигурация nginx для SPA
4. **Настройка SSL** — получение и настройка Let's Encrypt сертификатов
5. **Настройка CI/CD** — GitHub Actions для автоматического деплоя

## Оценка сложности
Высокая (4-6 часов)

## Зависимости
- Raspberry Pi 4 с доступом в интернет
- GitHub репозиторий для CI/CD
- Регистрация бесплатного домена
